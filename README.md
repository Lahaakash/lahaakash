## Projects I’ve worked on

### Adaptive Statistical Arbitrage Engine 
( Check Out: https://github.com/Lahaakash/stat-arb-engine )

This is the most challenging project I’ve worked on so far.

I built a full pipeline for a pair trading strategy using NSE stock data, starting from raw data all the way to signal generation and backtesting.

Some key things I worked on:

* Modeling stock relationships using a **Kalman Filter** instead of static assumptions
* Using a **Hidden Markov Model (HMM)** to understand when the market is stable vs unstable
* Building a backtesting setup with **walk-forward validation and Monte Carlo simulation**

Right now, I’m focused on debugging and improving it — especially making the results more reliable and realistic.

### EEG Signal Classification (Stress Detection)

Here I worked on processing real-world signal data.

* Built a preprocessing pipeline with filtering
* Used **CEEMDAN decomposition** to extract features from EEG signals
* Converted signals into images (GASF) and trained a CNN model

This project helped me understand how messy real data can be and how important preprocessing is.
